pub trait Spanned {
    fn span(&self) -> Span;
}

A trait that can provide the Span of the complete contents of a syntax tree node.

This trait is automatically implemented for all types that implement ToTokens from the quote crate.

See the module documentation for an example.

This trait is available if Syn is built with both the "parsing" and "printing" features.

Required Methods

fn span(&self) -> Span

Returns a Span covering the complete contents of this syntax tree node, or Span::call_site() if this node is empty.
